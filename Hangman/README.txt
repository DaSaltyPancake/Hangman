                           **THIS PROGRAM ONLY RUNS PROPERLY ON MAC OS**

INSTALLATION STEPS:

1) Ensure that the Hangman folder is in your Desktop

2) Download the Latest Version of Python 3 Here: 
https://www.python.org/downloads/

3) Hit Command and Spacebar to Open Spotlight Search

4) Input "Terminal" and Open the Application

5) Type the Following Commands and Hit Enter After Each:

pip3 install colorama
pip3 install playsound
pip3 install PyObjC

6) Enter these commands to run the program in your Terminal:

cd Desktop
cd Hangman-main
cd Hangman
python3 Hangman.py

7) Have Fun!
